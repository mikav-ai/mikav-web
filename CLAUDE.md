# Mikav Web — Claude Instructions

@AGENTS.md

## Additional Context

- Mikav is an open-source AI copilot and open Malayalam model built for Kerala's creative and cultural ecosystem
- Open datasets and models that individuals can explore and businesses and developers can build on
- The web app (`mikav-web`) is the primary user-facing interface

## Working on this project

1. Read `AGENTS.md` for conventions and architecture
2. Check `ROADMAP.md` for current priorities
3. Follow `CONTRIBUTING.md` guidelines for code style
4. Run `npm run build` to verify changes compile
5. Run `npm run lint` before committing

## Key paths

- `app/console/` — Main console UI (chat, settings, help)
- `components/console/` — Console shell components (layout, sidebar, header)
- `components/ui/` — shadcn/ui primitives
- `lib/utils.ts` — Shared utilities
- `app/globals.css` — Theme variables (light-only, primary: #c8242b)
